# v1 Roadmap

## Milestone 1: Workspace and models

- Rust workspace scaffold
- shared domain types
- CLI and helper entrypoints
- initial docs and ADRs

## Milestone 2: Real Windows probing

- replace placeholder probe data with real Windows inspection
- detect UEFI, GPT, Secure Boot, ESP, and BitLocker state
- convert unsupported states into hard blockers

## Milestone 3: Plan persistence and helper contract

- persist execution plans
- define the helper request/response contract
- make apply consume a concrete plan artifact

## Milestone 4: WinPE execution path

- stage managed WinPE artifacts
- register additive managed boot entry
- verify and roll back the WinPE path

## Milestone 5: Linux ISO profiles

- implement first supported distro profiles
- stage extracted boot artifacts
- verify and roll back the Linux path

## Milestone 6: Hardening

- checkpoint persistence
- interrupted-run recovery
- hardware validation
- packaging and operator guidance
